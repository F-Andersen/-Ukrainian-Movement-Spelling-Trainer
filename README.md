# Тренажер з правопису української мови

Графічний настільний застосунок на Python для тренування правопису української мови у форматі тестування з вибором варіанта відповіді.  
Програма генерує набір завдань, контролює час проходження тесту, автоматично перевіряє відповіді, обчислює відсоток успішності та веде рейтинг результатів.

## Основні можливості

- інтерактивний тест з фіксованою кількістю запитань (наприклад, 15 по 5 типах завдань);
- випадкове формування набору запитань із банку;
- таймер тесту (за замовчуванням 5 хвилин);
- автоматичний підрахунок балів та відсотка правильних відповідей;
- текстова інтерпретація рівня правопису (високий / достатній / середній / потрібне додаткове тренування);
- перегляд детальних відповідей по кожному запитанню;
- рейтинг (топ-20 результатів) із збереженням у локальному файлі;
- простий інтерфейс на основі `tkinter`, що не потребує окремого веб- або БД-сервера.

## Стек технологій

- **Мова програмування:** Python 3
- **GUI:** `tkinter`, `ttk` (стандартна бібліотека Python)
- **Збереження результатів:** JSON-файл (модуль `json`)
- **Робота з часом та датою:** `datetime`
- **Робота з випадковими значеннями:** `random`

Зовнішніх залежностей (pip-пакетів) застосунок не вимагає.
